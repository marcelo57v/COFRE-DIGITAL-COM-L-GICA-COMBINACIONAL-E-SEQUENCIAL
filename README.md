# Cofre Digital - Lógica Combinacional e Sequencial

## Descrição do Projeto
Este repositório contém o desenvolvimento de um **Cofre Digital**, realizado como parte da disciplina de Circuitos Digitais. O projeto aplica conceitos fundamentais de eletrônica digital, utilizando lógica combinacional para o processamento de dados e lógica sequencial para o controle de estados.

## Especificações Técnicas
O sistema foi projetado para executar as seguintes funções:
* **Lógica Combinacional:** Responsável pela verificação da senha, comparando a entrada do usuário com a chave armazenada através de comparadores de magnitude e portas lógicas.
* **Lógica Sequencial:** Gerencia os estados do cofre (Bloqueado, Aberto, Erro) utilizando Flip-Flops e registradores para persistência da senha e contagem de tentativas.

## Principais Componentes
* **Registradores:** Para armazenamento da senha digital.
* **Comparadores:** Para validar a entrada em relação ao segredo gravado.
* **Máquina de Estados (FSM):** Para controlar as transições do cofre baseadas em sinais de clock e inputs.
* **Displays:** Lógica de decodificação para displays de 7 segmentos para feedback visual.

## Estrutura do Repositório
